Quelques projets réalisés

📊 Dashboard de suivi de projets – Power BI, DAX

🔗 Voir sur GitHub

Conception d’un tableau de bord interactif dans Power BI pour le suivi de la performance des projets d’entreprise (coûts, délais, livrables), avec alertes visuelles en cas d’écart > 15 %. Le rapport est adapté à différents profils utilisateurs (directeurs généraux, régionaux et chefs de projet), permettant une lecture personnalisée des données, un pilotage stratégique efficace, et une détection rapide des projets en difficulté.

🌍 Étude de marché – Exportation de poulet bio

🔗 Voir sur GitHub

Analyse stratégique du marché pour une entreprise avicole souhaitant exporter du poulet biologique. Collecte de données (FAO, open data), analyse exploratoire, sélection des variables pertinentes, réduction de dimension par ACP, puis segmentation des pays cibles via K-means et classification hiérarchique. Résultat : identification de groupes de pays à fort potentiel pour orienter la stratégie d’exportation.

🧠 Détection de faux billets – Machine Learning

🔗 Voir sur GitHub

Développement d’une application de détection automatique de faux billets à partir de caractéristiques géométriques. Analyse exploratoire approfondie (distribution, ACP), complétion des données manquantes via régression linéaire, puis sélection des variables influentes. Entraînement de plusieurs modèles de classification (régression logistique, KNN, forêt aléatoire) avec validation croisée. Évaluation comparative via F1-score, recall, accuracy, taux de faux positifs/négatifs, puis déploiement du modèle final dans une application fonctionnelle.

⚖️ Automatisation de l’indicateur égalité F/H – KNIME

🔗 Voir sur GitHub

Projet réalisé à partir de données RH d’entreprise. Anonymisation préalable des données pour respecter le RGPD, puis automatisation du traitement avec KNIME : nettoyage, création de variables explicatives, calcul de l’indice global d’égalité professionnelle F/H. Le workflow génère automatiquement des visualisations et un tableau de bord synthétique pour le pilotage de l’égalité femmes-hommes.
